Array API#

template<typename ValueType, size_t Size, typename = std::enable_if_t<(Size > 0)>, typename = std::enable_if_t<std::is_default_constructible_v<ValueType>>>
class Array#

Fixed-size array container with compile-time size.

@requires ValueType must be default constructible

Complexity guarantees:

  • operator[]: O(1)

  • add(): O(1)

  • size(): O(1)

  • empty(): O(1)

  • begin(), end(): O(1)

  • cbegin(), cend(): O(1)

Template Parameters:

  • ValueType – The type of elements stored in the array

  • Size – The fixed size of the array, must be > 0
