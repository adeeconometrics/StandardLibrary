class Array#

Fixed-size array container with compile-time size.

@requires ValueType must be default constructible

Complexity guarantees:

  • operator[]: O(1)

  • add(): O(1)

  • size(): O(1)

  • empty(): O(1)

  • begin(), end(): O(1)

  • cbegin(), cend(): O(1)

Template Parameters:

  • ValueType – The type of elements stored in the array

  • Size – The fixed size of the array, must be > 0
